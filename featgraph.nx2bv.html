featgraph.nx2bv module

Conversion utilities from networkx to BVGraph

featgraph.nx2bv.make_asciigraph_txt(graph: Graph, path: str, encoding='utf-8', overwrite: bool = False)

Write the text file of adjacency lists (ASCIIGraph)

Parameters:

  • graph (Graph) – Networkx graph object

  • path (str) – Destination text file path

  • encoding – Encoding for the output file. Default is "utf-8"

  • overwrite (bool) – If True, then overwrite existing destination file

featgraph.nx2bv.make_attribute_txt(graph: Graph, path: str, attr: str, missing='', encoding='utf-8', overwrite: bool = False)

Write the text file for a node attribute

Parameters:

  • graph (Graph) – Networkx graph object

  • path (str) – Destination text file path

  • attr (str) – Attribute key

  • missing – Value to print when attribute is missing

  • encoding – Encoding for the output file. Default is "utf-8"

  • overwrite (bool) – If True, then overwrite existing destination file

featgraph.nx2bv.nx2bv(graph: Graph, bvgraph_basepath: str, attributes: Dict[str, Sequence[str]] | None = None, missing='', encoding='utf-8', overwrite: bool = False)

Convert a networkx graph to a BVGraph

Parameters:

  • graph (Graph) – Networkx graph object

  • bvgraph_basepath (str) – Base path for the BVGraph files

  • attributes (dict) – Attribute names and file suffix for export to text file. For each key/value pair, a text file is exported by printing for each node the value associated to the key in a file which path is derived from bvgraph_basepath using the value as suffix

  • missing – Value to print when node attribute value is missing

  • encoding – Encoding for the output files. Default is "utf-8"

  • overwrite (bool) – If True, then overwrite existing destination file
