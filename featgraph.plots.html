Plot the ROPE probabilities as a RGB matrix

Parameters:

  • df (pd.DataFrame) – ROPE probabilities dataframe

  • names (sequence of str) – Names of the different populations. If None, the names are inferred from the dataframe

  • order (callable) – Key function for sorting populations. Alternatively, if "auto", the populations are sorted by decreasing difference of average probabilities above and below the ROPE

  • y_label (str) – Column name for second population names. Default is "y"

  • probs_labels (triplet of str) – Column names for the probabilities below, within and above the ROPE

  • th (float) – If specified, threshold hypoteses at the given value

  • normalize (bool) – If True (default), normalize the RGB values. This results in brighter colors and better intelligibility

  • legend (bool) – If True, plot a legend on the axis

  • legend_selectors (sequence of int) – Specifies which legend entries to keep

  • ticks (bool) – If True (default), draw ticks and tick labels on both the x and y axis
