featgraph.pathutils module

Utilities for handling file paths

featgraph.pathutils.derived_paths(f: str, sep: str = '.') → Callable[[str], str]

Returns a helper function for generating derived paths

Parameters:

  • f (str) – Base path

  • sep (str) – Separator between base path and extension(s)

Returns:

Helper function

Example

>>> from featgraph.pathutils import derived_paths
>>> dbpath = derived_paths("db")
>>> dbpath()
'db'
>>> dbpath("csv")
'db.csv'
>>> dbpath("readme", "md")
'db.readme.md'
featgraph.pathutils.notisfile(f: str, func: ~typing.Callable | bool = <function isfile>, msg: str = "Found '%s'. Skipping", log: bool = True)

Check if file does not exist. If file is found, log a message

Parameters:

  • f (str) – File path to check

  • func (bool or callable) – If a boolean, then use this as the truth value for the file existence. Otherwise, call this function to get the truth value. Defaults to os.path.isfile()

  • msg (str) – Log message. It should have a string formattable field where the file name will be interpolated

  • log (bool) – If True, then log a message if the file is found

Example

>>> from featgraph.pathutils import notisfile
>>> notisfile("inexistent.5619hf")
True
>>> notisfile("inexistent.5619hf", True)
False
featgraph.pathutils.notisglob(f: str, func: ~typing.Callable | bool = <built-in function len>, **kwargs)

Check if no file that matches the expression exists. If a file is found, log a message

Parameters:

  • f (str) – File pattern to check

  • func (bool or callable) – If a boolean, then use this as the truth value for the file existence. Otherwise, call this function to get the truth value. Defaults to len()

  • kwargs – Keyword arguments for notisfile()

Example

>>> from featgraph.pathutils import notisglob
>>> notisglob("/path/to/nothing/*.ivsiva")
True
