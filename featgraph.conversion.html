featgraph.conversion module

Conversion functions for generating human-readable and BVGraph files from the original pickled dataset

featgraph.conversion.compress_to_bvgraph(dst: str, src: str | None = None, overwrite: bool = False)

Compress a text file of adjacency lists (ASCIIGraph) into a BVGraph

Parameters:

  • dst (str) – Destination BVGraph file basepath

  • src (str) – Source text file basepath. If None, then use the same basepath as dst

  • overwrite (bool) – If True, then overwrite existing destination file

featgraph.conversion.main(*argv)

Run conversion script

featgraph.conversion.make_asciigraph_txt(dst: str, src: str, idf: str, it: Callable[[Iterable], Iterable] | None = None, encoding='utf-8', overwrite: bool = False)

Write the text file of adjacency lists (ASCIIGraph)

Parameters:

  • dst (str) – Destination text file path

  • src (str) – Source pickle file

  • idf (str) – Graph node ids text filepath

  • it (callable) – Iterator wrapper function. If not None the adjacency lists iterator will be wrapped using this function. Mainly intended for use with tqdm

  • encoding – Encoding for output files. Default is "utf-8"

  • overwrite (bool) – If True, then overwrite existing destination file

featgraph.conversion.make_metadata_txt(dst: str | Callable, src: str, idf: str, it: Callable[[Iterable], Iterable] | None = None, labels: Iterable[str] | None = None, ext: str = '.txt', encoding='utf-8', overwrite: bool = False) → List[str]

Write the metadata text files

Parameters:

  • dst (str) – Destination text file basepath

  • src (str) – Source pickle file

  • idf (str) – Graph node ids text filepath

  • it (callable) – Iterator wrapper function. If not None the adjacency lists iterator will be wrapped using this function. Mainly intended for use with tqdm

  • labels (iterable of str) – Labels for which to write a file. If None (default), then write all metadata files

  • ext (str) – Common file extension. Default is ".txt"

  • encoding – Encoding for output files. Default is "utf-8"

  • overwrite (bool) – If True, then overwrite existing destination file

Returns:

Output file paths

Return type:

list of str
